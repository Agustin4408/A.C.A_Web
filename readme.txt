 Descripción en Español

Este proyecto es el sitio web oficial desarrollado para la comunidad de **Acción Católica de la Parroquia Nuestra Señora de Fátima** (González Catán). 

El objetivo es brindar un espacio digital moderno donde la comunidad pueda informarse sobre los grupos (Aspirantes, Jóvenes, Adultos), ver las novedades y acceder a cronogramas de eventos importantes como el Pesebre Viviente.
 Características Principales

Diseño Totalmente Responsivo: Se adapta perfectamente a celulares, tablets y computadoras (Mobile First).
Interfaz Moderna (UI/UX): Uso de paleta de colores institucionales (Azul, Celeste, Blanco), tipografías *Montserrat* y *Open Sans*, y efectos de vidrio (glassmorphism).
Animaciones CSS: Efectos de aparición al hacer scroll (Scroll Reveal) y transiciones suaves.
Carrusel Infinito: Galería de fotos con desplazamiento automático continuo.
Páginas Dedicadas:
    * `index.html`: Página de inicio con presentación institucional.
    * `participar.html`: Información detallada de grupos, requisitos y ubicación.
    * `noticias.html`: Blog de novedades.
    * `pesebre.html`: Página especial tipo "Landing Page" para el evento del Pesebre Viviente con cronograma detallado.
Navegación Móvil: Menú hamburguesa personalizado y optimizado.

 Tecnologías Utilizadas

HTML5: Estructura semántica.
CSS3: Flexbox, CSS Grid, Animaciones (@keyframes), Variables CSS.
JavaScript (Vanilla): Lógica para el menú móvil, barra de progreso de lectura y observadores de scroll.
Font Awesome: Íconos vectoriales.
Google Fonts: Tipografía.

English Description

This project is the official website developed for the Catholic Action community of Our Lady of Fatima Parish (González Catán).

The goal is to provide a modern digital space where the community can learn about the different groups (Aspirants, Youth, Adults), check the latest news, and access schedules for major events like the Living Nativity scene.

 Key Features

Fully Responsive Design: Adapts perfectly to mobile phones, tablets, and desktops (Mobile First approach).
Modern UI/UX: Uses institutional color palette (Blue, Light Blue, White), *Montserrat* and *Open Sans* typography, and glassmorphism effects.
CSS Animations: Scroll Reveal effects and smooth transitions.
Infinite Carousel: Photo gallery with continuous automatic scrolling.
Dedicated Pages:
    * `index.html`: Home page with institutional presentation.
    * `participar.html`: Detailed information on groups, requirements, and location.
    * `noticias.html`: News blog.
    * `pesebre.html`: Special "Landing Page" for the Living Nativity event with a detailed timeline.
Mobile Navigation: Custom optimized hamburger menu.

 Technologies Used

HTML5: Semantic structure.
CSS3: Flexbox, CSS Grid, Animations (@keyframes), CSS Variables.
JavaScript (Vanilla):Logic for mobile menu, reading progress bar, and scroll observers.
Font Awesome: Vector icons.
Google Fonts: Typography.
